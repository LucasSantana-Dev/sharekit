# Example Team Boards

## Cross-layer feature

| Track | Owner | Output | Handoff |
|---|---|---|---|
| API | Agent A | Contract and handler changes | Frontend can consume new response |
| UI | Agent B | Component and route updates | Review once API contract is stable |
| Tests | Agent C | Integration and regression coverage | Run after API and UI merge |
| Lead | Agent L | Final synthesis and report | Owns validation and summary |

## Competing fix paths

| Track | Owner | Output | Handoff |
|---|---|---|---|
| Hypothesis A | Agent A | Proposed fix and evidence | Compare against Hypothesis B |
| Hypothesis B | Agent B | Proposed fix and evidence | Compare against Hypothesis A |
| Review | Agent R | Tradeoff analysis | Lead decides final path |
